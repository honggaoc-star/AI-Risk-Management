# Objective and Drift Detection

This directory contains the objective-and-design-drift research program developed within AI Risk Management.

The research program defines a bounded problem, its relationship to the broader program, the relevant literature, a proposed design or analytical framework, and an evaluation strategy.

## Repository Sections

- [Research Exploration](./Research-Exploration/) — investigates unacknowledged divergence from user-authorized project state during extended human–AI collaboration and the feasibility of a specialized, low-friction monitoring model.
- [Research Notes](./Research-Notes/) — contains conceptual and exploratory notes supporting the research program.

Additional streams should be added only when they have a distinct research question and sufficient conceptual development.
